# CLI Commands

Pyohwa provides three main commands.

## `pyohwa init`

Create a new project:

```bash
pyohwa init <project-name>
```

## `pyohwa build`

Build the static site:

```bash
pyohwa build --root .
```

## `pyohwa dev`

Start the development server:

```bash
pyohwa dev --port 3000 --open
```
