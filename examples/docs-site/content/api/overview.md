# API Overview

Pyohwa provides a simple configuration-driven API through `pyohwa.toml`.

## Configuration Sections

- **site** — Title, description, base URL
- **build** — Content directory, output directory
- **theme** — Theme name, syntax highlighting
- **search** — Enable or disable search
- **seo** — Sitemap, RSS feed, OG image
