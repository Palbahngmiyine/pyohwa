# Installation

Install Pyohwa using one of the following methods.

## From Cargo

```bash
cargo install pyohwa-cli
```

## From Source

```bash
git clone https://github.com/example/pyohwa
cd pyohwa
cargo build --release
```

## Verify Installation

```bash
pyohwa --version
```

You should see the current version printed to the terminal.
